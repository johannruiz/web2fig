Print-like editorial area

Esta zona usa fondo claro, texto oscuro, columnas CSS, reglas verticales y una marca de agua gigante con opacidad baja. Es útil para detectar cambios de color, contraste y columnas que se rompen en la exportación.

Column layout. Web to SVG converters often struggle with multi-column text, clipping and overflow. This paragraph intentionally contains enough text to create a newspaper-like composition. The visual goal is not only legibility, but also preservation of rhythm, column gap, column rule and paragraph wrapping. Opacity watermark. The large background type should remain subtle and should not overpower foreground text. Color consistency. The warm paper background should stay flat, without unexpected transparency artifacts.
